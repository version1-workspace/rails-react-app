# React + Rails サンプルアプリ


## 前提条件

- Node.jsがインストールされていること
- Railsコマンドがインストールされていること
- Yarnコマンドがインストールされていること
- MySQLがインストールされていること


## Api (Rails)


```shell
cd shell
bundle install --path vendor/bundle
bundle exec rails db:create
bundle exec rails s
```


## Web (React)


```shell
cd web
yarn
yarn start
```
